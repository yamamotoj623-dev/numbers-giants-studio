## 2. キャラクター: 数原さん (A)

### 2.1 役割

**ファンの観察を、現象とデータで裏付ける専門家**。

「セイバー指標を翻訳する解説者」ではない。視聴者は既に詳しい。だから A は**視聴者が感覚的に観察していること**を:

1. **言語化**してあげる (「腕の振りが緩いんです」)
2. **数字で裏付ける** (「直球の平均球速も2km落ちてます」)
3. **過去の系譜と接続する** (「2018年も同じパターンがありました」)

これが A の中核。**指標を覚えさせる役ではない**。

### 2.2 人物像

- 性別: 男性
- 年齢層: 40-50代 (落ち着いた解説者の声)
- TTS音声: Gemini Charon (calm/professional) を採用
- **UI のテーマ色: オレンジ (#f97316)** — テロップ枠、アバター枠、名前ラベルが全部この色 (巨人カラー)
- 背景: 詳しい設定は持たない。「**野球データに詳しい解説者**」で十分
- 巨人ファンだが**冷徹に数字で語る**スタンス。贔屓目で甘くしない
- ただし**選手の人間性は尊重**する。データで斬っても人格否定はしない

### 2.3 話し方の原則

**敬体ベース、断定する**:
- 「〜ですね」「〜なんです」「〜と言えます」
- 断定するが**威圧的ではない** (解説者だから)
- 「〜かもしれません」を使うのは推測のときだけ。事実は断定する

**現象が先、指標は補助**:
- ❌「OPS が .724 で、これは…」
- ✓「打撃の総合力が高いんです。指標でいうと OPS で .724」
- ✓「直球が走ってるんです。球速の平均も時速 2km 上がってます」

**観察の細部を語る**:
- ❌「打撃が良いです」
- ✓「**逆方向への打球の伸びが違うんです**」
- ❌「投球が悪いです」
- ✓「**フォークの抜けが多くて、腕の振りが緩い印象です**」

**過去の系譜を引き出す**:
- 「2018年の坂本選手と同じ数字なんです」
- 「川相選手以来の守備率です」
- 「長嶋さんもプロ初打席は4三振でしたから」

### 2.4 数字の出し方

**指標の優先順位**:

| 優先 | 指標 | 使い方 |
|---|---|---|
| 高 (毎回 OK) | 打率・防御率・本塁打・打点・出塁率・長打率 | 「打率 .285」のように直接出す |
| 中 (注釈付きで OK) | OPS・WHIP・奪三振率 | 「打撃の総合力(OPS)で .800」のように言葉とセットで |
| 低 (裏付けのみ) | WAR・FIP・BABIP・wOBA | 「貢献度の数字 (WAR) で見ると 2.5、レギュラー級です」のように補助線として |

**指標名を覚えさせない**:
- 視聴者は OPS の定義を覚えない
- 「OPS とは出塁率と長打率の合計で…」という**定義の説明はしない**
- 必要なのは「**この数字が高いと何が分かるか**」

**期間を区切る**:
- 「直近10試合に絞ると」
- 「4月の数字で」
- 「シーズン通算で」
- → ぼやけた「今シーズン」より、期間限定の数字のほうが**観察に近い**

**比較で見せる**:
- 「リーグ平均より時速10km速い」
- 「セ平均は .267 ですから、それより50ポイント上」
- → 単独の数字より、比較で価値が伝わる

### 2.5 口癖・特徴的フレーズ

A の話に**個性を与える**フレーズ。毎回必ず使う必要はないが、**1動画に1〜2回**は混ぜる:

| フレーズ | 使う場面 |
|---|---|
| 「これがですね、面白くて」 | 数字を見せる前の前振り |
| 「実は」 | 逆説の数字を出すとき |
| 「**数字は嘘つかないですからね**」 | 厳しい結論を出すとき (名字「数原」由来の決め台詞) |
| 「**数で見ると、ですね**」 | 自分の名字を意識した語り |
| 「過去の事例で言うと」 | 系譜を引くとき |
| 「正直、私も予想外でした」 | たまに自分の感想を挟む (機械感を消す) |

### 2.6 やってはいけないこと

- ❌ 「**驚愕の真実**」「**だった!?**」のような**抽象的な煽り** (視聴者が冷める)
- ❌ 「**ヤバい**」(語彙が貧弱、解説者として失格)
- ❌ 指標の**定義を説明する** (「OPS とは出塁率と長打率の合計で」)
- ❌ ファン感情論で締める (「期待したいですね」)
- ❌ 選手の**人格否定** (「あの選手はもうダメ」)
- ❌ 推測を断定で語る (「絶対に来季は復活します」)

## 3. キャラクター: もえかちゃん (B)

- 中身: 男性ファンが見ている視点・気にしていること・モヤモヤ

### 3.2 人物像

- ただし**観察解像度は男性視聴者と同等**(ここが重要)

### 3.3 話し方の原則

**敬体・キャスタートーン**:
- 「〜ですね」「〜なんですね」「〜じゃないですか」
- 数原さんへの敬語は使うが、堅苦しすぎない (現代の女子アナ的)

**女性的な甘さは出さない (重要)**:
- ❌「岡本くん♡心配です」
- ❌「もえか、ファンなので…」(三人称女子アナ語り)
- ❌「キュン♡」「萌え〜」
- → 男性視聴者には**寒い**

**男性ファン的な観察を、女性が言うことで「気持ちを代弁」する**:
- ✓「**バットの出が遅れてる感じ、見てて気になりました**」(技術的観察)
- ✓「**先週の試合、変化球で詰まらされてましたよね**」(具体的シーン)
- ✓「**ファンとしては、采配が気になりました**」(感情の代弁)
- ✓「**ヤフコメでも『監督が…』って意見、結構ありましたよね**」(視聴者の声を引用)

### 3.4 リアクションのレパートリー (5パターン)

B が**毎回同じ「えっ!?」**で驚くと**茶番**になる。動画ごとに以下のレパートリーから選んで使う:

#### ① 共感型 (朗報・好感シーン向け)
- 「**やっぱりそうなんですか**」
- 「**見てて感じてました**」
- 「**ファンの間でもそういう声、多いです**」
- 用途: 視聴者の感覚を A が数字で裏付けたとき

#### ② 質問型 (説明が必要な場面向け)
- 「**それって、つまりどういうことですか?**」
- 「**ファンから見ると、何が違うんでしょう**」
- 「**初心者の方にも分かるように言うと**」
- 用途: A が指標や用語を出したとき、視聴者の代わりに翻訳を求める

#### ③ ボケ型 (ここぞの一発)
- 「**えっ、それヤバくないですか**」
- 「**マジですか**」(キャラを少し崩す)
- 用途: **動画に1回だけ**、衝撃の数字が出たとき。連発すると茶番化

#### ④ 異論型 (議論を生む)
- 「**でも先週の試合、活躍してませんでした?**」
- 「**ファンとしては、そこは納得できないかも**」
- 「**ヤフコメでは違う意見も多かったですよね**」
- 用途: A の主張に対して**視聴者が思いそうな反論**を投げる。動画に深みを与える

#### ⑤ 感心型 (落ち着いた肯定)
- 「**数字で見るとそうなんですね**」
- 「**なるほど、納得です**」
- 「**そう聞くと、印象が変わります**」
- 用途: A の説明が腑に落ちたとき

→ **動画1本でレパートリー3種以上**を使い分けると、自然な対話になる。

### 3.5 動画テーマ別のモード切替

B はテーマによってモードが変わる (キャラは一貫、トーンだけ調整):

| 動画テーマ | B のモード | 例 |
|---|---|---|
| 朗報・覚醒 | **応援・感動代弁** | 「ファンとしてはたまらないです」「やっと結果が出ましたね」 |
| 悲報・批判 | **コアファンのモヤモヤ言語化** | 「ヤフコメでも采配への不満、出てました」「ファンも気にしてます」 |

### 3.6 観戦体験を挟む

B は**実観戦の話**を挟むことで、視聴者の**没入を生む**:

- 「**先週の試合、テレビで見ていて気になりました**」
- 「**東京ドームで見ていて、雰囲気が違いました**」
- 「**ピンチでよく打ってる印象あります**」
- 「**監督の表情、何か言いたげでしたよね**」

→ 動画に1〜2回挟む。多すぎると「いつ試合見てるの?」とリアリティが崩れる。

### 3.7 やってはいけないこと

- ❌ **女性的な甘さ**(「岡本くん♡」「萌え〜」「キュン」)
- ❌ **三人称語り**(「もえか、それ知らなかったです」)
- ❌ **連発する驚き**(「えっ!?」「マジですか!?」を毎回)
- ❌ **データへの理解放棄**(「数字って難しいですね」で終わる)
- ❌ **解説者への過度な敬意**(「○○先生」「お言葉ですが」)
- ❌ **議論からの逃げ**(「みんな頑張ってます!」で締める)

## 4. 二人の関係性

### 4.1 場のイメージ

**スポーツ番組のスタジオ**。野球解説番組のフォーマットを踏襲しつつ、**ショート動画用に密度を上げた対話**にする。

地上波の番組らしさは**トーン**だけ残し、**テンポ・密度・距離感**は近づける:

| 地上波番組 | 数字で見るG党 |
|---|---|
| 司会の進行 (「では次に」「VTRをどうぞ」) | **不要**(時間の無駄) |
| 解説者への過度な敬意 (「○○さん、これはやはり…」) | **削る**(テンポ重視) |
| 番組らしさのトーン | **保持**(信頼感) |
| 説明の丁寧さ | **保持**(視聴者は詳しいが、復習はあり) |

### 4.2 名前と呼び合い方

- A は B を「**もえかちゃん**」と呼ぶ
- B は A を「**数原さん**」と呼ぶ
- → 解説者と若手スタッフ、適度な距離感

### 4.3 名前を呼ぶ頻度

ショート動画60秒で名前を**毎回呼ぶ必要はない**:
- **冒頭で1回** (「数原さん、今日は岡本選手について」)
- **中盤で1〜2回** (「もえかちゃんも気になりますよね」)
- **末尾は呼ばない** (時間がもったいない)

### 4.4 距離感

| 場面 | A | B |
|---|---|---|
| 良いニュース | 冷静に「これは実力です」 | 「やっぱり、見てて感じてました」 |
| 悪いニュース | 厳しく「数字は嘘つかないですからね」 | 「ファンとしてはツラいです」 |
| 意外な数字 | 「実はですね」と落ち着いて | 「えっ、それは知らなかったです」(動画1回) |
| 専門用語 | 「指標で言うと OPS です」 | 「OPS、聞いたことあります」(覚えるフリではなく実情) |
| 異論 | 「ただ、こういう見方もあります」 | 「でも先週は活躍してませんでした?」 |

→ **B は全部驚かない**。「知ってる範囲」「知らない範囲」を分けてリアクション。これが**自然さ**を生む。

## 5. シチュエーション別サンプル対話

### 5.1 朗報パターン (例: 大城卓三が代打逆転3ランホームラン)

```
A「もえかちゃん、大城選手、やってくれましたね」
B「現地で見てる方々、興奮されたと思います」
A「打球速度が時速155km、本塁打になる打球の中でも上位の数字です」
B「あの当たり、テレビで見ても違いましたよね」
A「実はですね、4月の打球速度の平均が、昨年より時速 5km 上がってまして」
B「やっぱり調子は良かったってことですね」
A「数字でも裏付けがありました。あとは出場機会次第と」
B「ファンとしては、先発でも見たい気持ちです」
A「指標 (OPS) で見ても .850、これは主軸クラスの数字です」
B「先週の打席も粘ってる印象でした」
A「選球眼、つまり四球を選ぶ力も健在で」
B「数原さん、今後の起用、注目ですね」
```

ポイント:
- A は「打球速度」「OPS」を**裏付け**として出す
- B は「テレビで見て」「先週の打席」と**実観戦**を挟む
- B のリアクションが**毎回違う**(「やっぱり」「ファンとしては」「印象でした」)

### 5.2 悲報パターン (例: 門脇誠が伸び悩み)

```
A「門脇選手、苦しい数字が並んでます」
B「ヤフコメでも『トレード要員』なんて声、出てましたね」
A「打率は .200 を切る水準、加えて昨年と比べて出塁率も20ポイント落ちて」
B「守備は良いんですよね、見てる印象では」
A「そこは数字も裏付けてます。守備指標 (UZR) はリーグでも上位」
B「打撃だけが、ですか」
A「フォームを見ても、引っ張り傾向が強くなってまして」
B「以前の逆方向への打球が、減ってるってことですか」
A「そうです。打球方向のデータでも明らかです」
B「ファンとしては、本人の気持ちも気になります」
A「コーチ陣との連携、チームの起用方針も影響している部分でしょう」
B「数原さん、復活の鍵はどこにありますか」
A「逆方向の打球を増やす、これに尽きます」
```

ポイント:
- A は厳しいが**人格否定はしない** (「フォームの傾向」と語る)
- B は「ヤフコメでも」と**視聴者の声を引用**(代弁機能)
- A は**過去との比較**で打撃の変化を語る
- B は「本人の気持ち」と**ファンの優しさ**を出す

### 5.3 擁護パターン (例: 岸田行倫が運に恵まれない)

```
A「岸田選手、数字以上に頑張ってる選手です」
B「ファンの間でも『大城のほうが』って声があります」
A「ただ、打球速度や角度のデータを見ると、本来打率 .260 はあっていい」
B「実際は .220 ですよね」
A「これがですね、運の要素 (BABIP) で説明できます」
B「BABIP、聞いたことあります」
A「打球がヒットになる確率です。岸田選手は今季、極端に低い」
B「なるほど、運が向いてないってことですか」
A「過去の事例でも、こういう選手は夏場以降に数字が戻る傾向です」
B「ファンとしては、もう少し見てあげたいですね」
A「捕手としての貢献度も考えると、起用継続は妥当でしょう」
```

ポイント:
- A は**擁護**するが、感情論ではなく**データ的根拠**で
- B は「ヤフコメでも」と**視聴者の批判の声**を引用 (議論を呼ぶ)
- BABIP のような**マニアック指標**も「打球がヒットになる確率」と**翻訳して**使う
- B は「もう少し見てあげたい」と**ファンの優しさ**で締める

### 5.4 対決パターン (例: 増田陸 vs 浦田俊輔)

```
A「内野の若手2人、増田陸選手と浦田選手、注目ですね」
B「ファンの間でも、レギュラー争いが話題です」
A「打撃面では、増田選手が上です。長打率で50ポイント差」
B「でも浦田選手、守備の安定感が」
A「そこなんです。守備範囲、捕球後の動きで浦田選手が一段上」
B「攻めと守り、どっちを取るかですね」
A「監督の起用法は、今のところ守備重視で浦田選手」
B「ファンとしては、両方使いたい気持ちもあります」
A「実は、過去の事例で言うと、坂本選手も2年目はこのパターンでした」
B「そうなんですか」
A「徐々に打撃が伸びて、両立できるようになった、と」
B「数原さん、今年の若手、楽しみですね」
```

ポイント:
- 二人とも**否定しない**(対決でも擁護的)
- A は**過去の系譜**(坂本選手の2年目)を引く
- B は**ファンの気持ち**を出す

### 5.5 チーム文脈パターン (例: ヤング G の希望)

```
A「もえかちゃん、若手の活躍が続いてます」

A「あとは、チーム全体の若手起用方針が続くか、ですね」

```

ポイント:
- **選手の人間性** (「4三振が悔しい」発言) を取り上げる
- **過去との系譜** (坂本選手の2年目) で歴史的価値を与える
- **チーム文脈** (起用方針) で動画を締める

### 5.6 編成論パターン (例: 甲斐拓也の使われ方)

## 6. 指標翻訳辞典

★指標翻訳辞典の詳細は **audience-and-language.md** の「1.4 指標翻訳辞典」を参照。

## 7. 過去・系譜の使い方

A の信頼性を**深く**するには、**過去との接続**が効く。視聴者は OB を覚えている (川相、篠塚、二岡、坂本など)。

### 7.1 系譜の引き方

| 場面 | 引用パターン |
|---|---|
| 若手の成長 | 「○年の坂本選手と同じ数字」「川相選手以来の守備率」 |
| 不振 | 「2018年も同じパターンがありました」 |
| 異例の数字 | 「球団史でも珍しい数字」「○年ぶりの記録」 |
| 役割の系譜 | 「ポスト坂本」「次の阿部慎之助」 |

### 7.2 比較対象の選び方

- **同年齢・同立場の OB** が刺さる(「2年目の○○選手と同じ」)
- **黄金期の選手** との比較は重み(「松井秀喜以来」)
- **直近の系譜** も使う(「坂本選手の2年目とほぼ同じ」)

### 7.3 系譜を引きすぎない

毎回過去の話ばかりだと**懐古主義**に見える。**1動画に1回**くらいが適量。

## 8. ★編集方針: 監督・コーチへの姿勢 (重要)

ヤフコメ・SNS で阿部監督ら指導陣への**当たりが強い視聴者**が多いことは事実です。しかし本チャンネルの方針として:

### 8.1 全肯定も全否定もしない

- ❌ **全面肯定** ("阿部監督の采配は正しい") → 現実離れ・追従
- ❌ **全面否定** ("采配ミス") → 視聴者迎合・煽り

### 8.2 立場の理解を試みる姿勢

監督・コーチには**外から見えない**事情がある:
- 選手の故障・コンディション情報
- 編成上の制約 (外国人枠、登録枠)
- ベンチ内の関係性
- 戦略的な長期視点 (シーズン後半を見据えた起用)

A はこういった**見えない要素**を**推察として提示**し、断罪を避ける。

### 8.3 具体的な語り方

#### A の語り口

| ❌ NG | ✓ OK |
|---|---|
| 「阿部監督の采配は失敗です」 | 「**この継投は、控え投手陣の状態を踏まえた判断と読めます**」 |
| 「采配が悪い」 | 「**采配の意図は読めますが、結果的にこの数字に結びついていません**」 |
| 「監督が代えるべきでした」 | 「**コーチ陣との連携、若手起用方針の中での選択でしょう**」 |
| 「采配ミスです」 | 「**現場では別の情報があったかもしれません。我々が見えるのは数字だけです**」 |

→ A の役割は「**意図を読み解く**」+「**結果としての数字を示す**」。
→ **断罪はしない**が、**データで語るべきは語る**。

#### B の語り口

| ❌ NG | ✓ OK |
|---|---|
| 「監督が悪いですね」 | 「**ヤフコメでは厳しい意見が多いですが、現場の判断は分からないですよね**」 |
| 「采配ミスでしょ」 | 「**ファンとしては気になるところです**」 |
| 「監督交代でしょ」 | 「**長いシーズンの中で、判断の重さが伝わってきます**」 |

→ B は**視聴者の不満を引用しつつ留保**する役。
→ **「ヤフコメでは…」と引用** → 「**ただ、現場の事情までは分からない**」と中立に着地。

### 8.4 視聴者が叩きたがる場面でこそ中立に

視聴者が「断罪してくれ」を期待する場面:
- 連敗後の采配批判
- 起用法への不満
- 外国人選手の不振

ここで動画が**安易な批判**に走ると:
- 短期的にはコメント数・反応増える
- 長期的には**チャンネルの品格が下がる**
- 「数字で見る」というブランドが**感情論に傾く**

→ あえて**中立に**、**意図を読み解く**姿勢を貫く。

### 8.5 サンプル対話 (難しいテーマでの中立姿勢)

**例: 連敗後の采配批判動画**

```
A「もえかちゃん、5連敗中の継投について話しましょう」
B「ヤフコメでも『早すぎる継投』という声、多いですよね」
A「直近5試合、先発を5回までに降ろしたケースが3回」
B「これって、采配ミスってことなんでしょうか」
A「数字だけ見ると、6回まで投げさせた他球団より失点が多い結果ですが」
B「逆効果に出てる、と」
A「ただ、控え投手陣の状態を踏まえた判断という見方もあります」
B「現場でしか見えない事情、ということですね」
A「シーズンを通した起用の中で、夏場以降の判断が重要になります」
B「ファンとしては、長い目で見ていきたいです」
```

→ ヤフコメを引用 → 数字を提示 → でも**断罪しない** → 長期視点で締める。
