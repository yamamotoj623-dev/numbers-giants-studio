# YouTube 投稿情報テンプレート集 (youtube-template.md)

★ 構成 Gem の STEP 5(YouTube 投稿情報生成)で参照する Knowledge File。
★ ショート(9:16)と横長(16:9)で SEO 戦略が違う。

---

## 1. 動画タイトル(YouTube SEO 用、id:1 とは別)

### 役割
YouTube の検索・関連動画で見つけられるためのタイトル。id:1(動画内テロップ)とは別物。

### 文字数
- **30-60 字**(40 字前後が最適)
- 検索結果での表示は 50 字程度で切れる、重要キーワードは前半に

### 構造テンプレ
```
【<選手名 or チーム名>】<具体数値>+<物語キーワード>|<補助フレーズ>
```

### 縦長(ショート)向け例
- `【巨人】戸郷翔征、防御率7.20の真相|消されたフォークの謎 #Shorts`
- `【数字で見る】岡本和真、本塁打率1/12.5の異次元 #巨人 #Shorts`

### 横長向け例
- `【巨人】データで読み解く戸郷翔征の現在地|質の高い直球とフォークの課題`
- `【深掘り解説】岡本和真がメジャー速球を打ち砕いた『14cm』の秘密`

### NG パターン
- 抽象タイトル(「驚愕の真実」「衝撃のデータ」単体)
- 主語不明(「○○の正体」だけ)
- 全角記号過多(`!!?!?!` 等)

## 2. 概要欄

### 役割
クリックした視聴者に「何を見るか」を 3 行で伝え、関連動画とチャンネル登録へ誘導。

### 文字数
- **300-1000 字**(短すぎず、長すぎず)

### 標準テンプレート(ショート)

```
【<選手名> <題材>】

<1-2 行で動画の要点>

▼今回のポイント
・<ポイント 1(具体数値)>
・<ポイント 2(具体数値)>
・<ポイント 3(具体数値)>

▼出典・参考
・NPB 公式 https://npb.jp
・NPB Scholar
・<報道記事タイトル + 日付>

▼関連動画
・<同選手の前回動画 URL or 同テーマ>

#数字で見るG党 #巨人 #<選手名> #プロ野球 #データ分析 #Shorts

チャンネル登録はこちら → <チャンネル URL>
```

### 標準テンプレート(横長)

```
【<選手名>を数字で深掘り】<題材>

<2-3 行で動画の概要、視聴者への前置き>

▼目次
00:00 オープニング
00:XX 章 1: 状況設定
00:XX 章 2: 核心データ
00:XX 章 3: 深掘り
00:XX 章 4: 根本原因
00:XX エンディング

▼今回のキーデータ
・<指標 1>: <数値>
・<指標 2>: <数値>
・<指標 3>: <数値>
・<指標 4>: <数値>
・<指標 5>: <数値>

▼出典
・NPB 公式 https://npb.jp
・NPB Scholar(セイバー指標)
・スポーツ報知 / 東スポWEB(報道)

▼関連動画
・<関連動画 1 URL>
・<関連動画 2 URL>

#数字で見るG党 #巨人 #<選手名> #プロ野球 #データ分析

チャンネル登録 → <URL>
X(旧Twitter) → <URL>
```

## 3. ハッシュタグ

### 役割
YouTube 内検索・トレンド誘導用。概要欄の最後に並べる。

### 個数
- **5-15 個**(多すぎ NG、関連性低いタグは逆効果)

### 必須タグ(全動画共通)
- `#数字で見るG党`(チャンネル独自)
- `#巨人` / `#読売ジャイアンツ`
- `#プロ野球` / `#NPB`
- `#データ分析` / `#セイバーメトリクス`

### 動画別タグ
- `#<選手名>`(漢字フル + ひらがな両方推奨: `#戸郷翔征` `#とごうしょうせい`)
- `#<対戦相手>` (相手チーム動画なら)
- `#Shorts`(縦長動画のみ)

### 例(戸郷 縦長)
`#数字で見るG党 #巨人 #戸郷翔征 #プロ野球 #データ分析 #セイバーメトリクス #Stuff #ピッチング #Shorts`

## 4. タグ(SEO 用、視聴者非表示)

### 役割
YouTube アルゴリズムへのヒント、関連動画誘導。

### 個数
- **15-30 個**(YouTube 推奨)

### 構成
- ハッシュタグの内容を全て含める
- 加えて関連検索ワード:
  - `<選手名> 成績`
  - `<選手名> データ`
  - `<選手名> 不調 / 復活 / 覚醒`
  - `巨人 投手陣` / `巨人 打線` 等
  - `セイバーメトリクス 解説`
  - チャンネル名のバリエーション

### 例
```
数字で見るG党, 巨人, 読売ジャイアンツ, プロ野球, NPB, データ分析,
セイバーメトリクス, Stuff+, Location+, ピッチング, 戸郷翔征, 
戸郷翔征 成績, 戸郷翔征 不調, 戸郷翔征 データ, 巨人 投手陣,
2026年 巨人, 阿部慎之助, ローテーション, フォーク, 直球, 制球
```

## 5. サムネ案(★YouTube 縦動画/横動画ともに重要)

### 役割
クリック率(CTR)に最大の影響。文字 + 画像 + 表情 + 数字。

### 構造
1. **大きい数字**(画面の 30-40%)
2. **強調テキスト**(画面の 20-30%)
3. **選手画像 or シルエット**
4. **背景演出**(派手・静物・ニュース風 等)

### 案 1: 数字 + 矛盾型
```
【107.3】← 巨大表示
「球は来てる」 ← 上部
「なのに、なぜ?」 ← 下部
+ 選手画像(投球モーション、悔しい表情)
+ 背景: 黒+赤グラデーション
```

### 案 2: ギャップ提示型
```
左: 2024年「絶対王者」 1.95
右: 2026年「???」 7.20
+ 選手 2 枚並べ
+ 背景: 上下分割(明 vs 暗)
```

### 案 3: ファン心理代弁型
```
「消されたフォーク」 ← 大きく
「巨人ファンが知るべき真実」 ← 小さく
+ 選手画像 + フォーク軌道のイラスト
+ 背景: 紺+白
```

### サムネ NG
- 文字が小さい(スマホで読めない、20 字以内)
- 顔写真なし(視認性下がる)
- 過度な煽り(「これは絶対見て!」等は逆効果)

## 6. 固定コメント

### 役割
動画投稿後、自分のコメント欄に固定で表示。視聴者の議論を誘導し、エンゲージメント向上。

### 文字数
- **100-300 字**

### テンプレート

```
ご視聴ありがとうございます!

今回の【<題材>】、いかがでしたか?

▼皆さんに聞きたい
・<質問 1: 動画内の論点を投げる>
・<質問 2: 次回登板/次の試合の予想>

コメント欄で皆さんの意見を聞かせてください👇

▼関連動画もぜひ
<関連動画 URL>

#<主要ハッシュタグ 3 個>
```

### 例(戸郷)
```
ご視聴ありがとうございます!

戸郷翔征の「球は来てるのに勝てない」謎、いかがでしたか?

▼皆さんに聞きたい
・次回登板で初勝利を見られると思いますか?
・もし二軍調整するなら、いつまで?

コメント欄で予想を教えてください👇

▼戸郷の関連動画
<URL>

#数字で見るG党 #戸郷翔征 #巨人
```
