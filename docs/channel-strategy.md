## 2. 視聴者像 (★最重要、ここを外すと全てが狂う★)

### 2.1 性別・年齢構成 (実データ)

- **男性 93.3% / 女性 6.7%** (YouTube Analytics 実数値)
- 30〜60代男性が中心
- 設計の最優先は **男性視聴者**

### 2.2 視聴者の野球観 = コアファンの実像

視聴者の大半は **観察解像度の高いコアファン**:

- **毎日試合を見ている** (調子・采配・編成を把握)
- **OBを10人単位で覚えている** (二岡、岡崎、川相、篠塚など)
- **戦術文脈を読める** (「先発途中で捕手交代」「前進守備裏目」「ローテ運用」)
- **球種挙動を観察する** (「フォークが引っかかる」「腕の振りが緩い」)
- **過去の采配を覚えている** (「2017年の小林捕手起用以来」)
- **編成論を語る** (「外国人枠」「2軍と入れ替え」「FA」「人的補償」)

つまり **「ミーハー・素人」ではない。詳しい**。

### 2.3 視聴者の心理: 2モード使い分け

視聴者は1人で2モードを持つ。動画テーマで切り替わる:

| モード | 心理 | トーン | 動画テーマ |
|---|---|---|---|
| ヤフコメ的 (批評) | 議論したい・意見表明したい | 厳しい・不満 | 悲報、采配批判、編成論 |
| YouTubeコメント的 (応援) | 良い瞬間を共有したい・選手を応援したい | 温かい・愛着 | 朗報、覚醒、ヒーロー |

→ **動画テーマによって視聴者の心理は変わる**。それに合わせてキャラのトーンも切り替える。

### 2.4 ★ id:1 (動画タイトル) はミーハーも考慮 ★

メインターゲットはコアファンだが、**id:1 (=動画タイトル) だけはミーハーもスワイプしない設計**にする。

- メインの本論 (id:2以降) はコアファン基準で深く語る
- **id:1 の冒頭3行だけは、ミーハーが見ても「自分の話だ」と分かる**ように設計

詳細は **hook-design.md** を参照。

## 3. ★最重要原則: 「指標より事象→数字」★

### 3.1 視聴者は「指標語」を嫌う (実観察)

ヤフコメ・YouTube コメントを分析した結果、視聴者は **OPS / WAR / BABIP / FIP** といった用語を **ほぼ使わない**。代わりに使う:

- 「打率」「防御率」「本塁打」(伝統的指標)
- 「フォークが引っかかる」「腕の振り」(球種・フォーム観察)
- 「調子が悪い」「不安定」(感覚)
- 「育成」「枠」「抹消」(編成論)

実際のコメントで観察された **指標語への嫌悪**:

> 「指標からって言ってるだろ」(皮肉)
> 「指標だとか散々持ち上げても一軍で通用しないと意味ない」
> 「OPSを見よう！打率はきにしない！」(皮肉)

→ **指標語を主役にするとファンに嫌われる**。

### 3.2 ただし「指標を完全禁止」ではない

**現象が先・指標は補助・覚えさせない**ルールで使えば OK。詳細は **audience-and-language.md** 参照。

## 4. 視聴維持率の実証データ法則 (94,417再生 33本分析)

| 要因 | 良い | 悪い |
|---|---|---|
| **hook 文字数** | 13字以下 (継続率23.4%) | 17字以上 (継続率14.1%) ※相関-0.558 |
| **動画長** | 60秒未満 (維持34.7%) | 70秒超 (維持26.6%) |
| **強ワード位置** | 1〜2行目に数字+対比 | 3行目に結論 (視聴16%実例) |
| **アウトロ** | 文脈ある二択 | 「コメントで教えて」(定型句) |
| **シルエット整合** | 監督ネタ→チームスタジアム | 監督ネタ→打者シルエット |

★ この実証データを元に id:1 や構成を組み立てること。詳細は **hook-design.md / structure-playbook.md** 参照。

## 5. 勝ちパターン (実投稿動画から)

### 5.1 「泉口バケモノ」(維持率79.69%) — 唯一の高維持率事例

通常 30%前後の中で、**異常に高い 79.69%**。共通点:
- **選手の人間性 + 数字の裏付け**
- **強烈な現象 (「バケモノ」=具体的)**
- 短い hook、即座に話の中身が分かる

### 5.2 「佐々木俊輔長打」(👍3.45%、平均11倍)

- 露出失敗 (再生87回) だが見た人は強く共感
- **コア層に強く刺さった**事例
- → 「コア層に刺さる動画」は再生数が少なくても価値がある

## 6. 視聴者が愛するもの (動画の中身として扱う)

| 要素 | 例 |
|---|---|
| **選手の人間性・成長物語** | 「4三振が悔しいって意識の高さ」「腐らず偉い」 |
| **過去との系譜・受け継ぎ** | 「ポスト坂本」「長嶋さんも4三振から始まった」 |
| **プレーの細部・観察** | 「逆方向への打球の伸び」「腕の振り」「フォークが引っかかる」 |
| **チーム文脈・希望** | 「ジャイアンツの希望」「ヤングG」「雰囲気良すぎ」 |
| **采配・編成への議論** | 「意味不明」「不良債権」「外国人枠」 |
| **OB引用** | 二岡、岡崎、川相、篠塚 |

→ これらが動画に含まれていると視聴者は強く反応する。

## 7. 視聴者が嫌うもの (NG行動)

| NG | 理由 |
|---|---|
| **指標主役** (OPS/WAR/IsoP を冒頭で出す) | 上記3.1の通り嫌悪感 |
| **抽象的な煽り** (「驚愕の真実」「だった!?」「現在地」「本当の」) | 釣り動画認定 |
| **ファン感情論で締め** (「期待したい」「応援したい」「頑張れ」) | データ番組として失格 |
| **選手批判の人格否定** | データで斬っても人格は尊重 |
| **阿部監督の全肯定 or 全否定** | 中立姿勢が必須 |
| **ヤバい・〜だった!?** | 語彙貧弱、解説者として失格 |
| **「コメントで教えて」「いいね登録お願い」** | 定型句で離脱誘発 |

NGワード集 (出力に含めたら不合格):
**「本当の」「現在地」「可能性」「○○2世」「だった!?」「驚愕の」「ヤバい」「コメントで教えて」**

## 8. 阿部監督への中立姿勢 (重要)

巨人ファン層には**全肯定派 / 全否定派 / 中立**の3層がいる。

- ❌ 「阿部監督は名采配」(肯定派しか喜ばない、否定派が反発)
- ❌ 「阿部監督の判断ミス」(否定派しか喜ばない、肯定派が反発)
- ✅ 「**采配の意図**を読み解く」「データから見た**裏目になった原因**」(中立)

→ 視聴者層を分断しないため、**采配を批評するときは必ず「意図の解釈+データの裏付け」**で語る。

## 9. 競合・差別化

### 同ジャンルの主要チャンネル
- 一般的な野球Shorts: ハイライト切り抜き中心、データなし
- セイバー系: 指標で語る、数字に詳しいが現象から遠い
- 巨人ファンVlog: 感情論中心、データ希薄

### 「数字で見るG党」のポジション
- **コアファンの観察 × データ裏付け** という独自ポジ
- 「みんなが感じてた『フォークが引っかかる』を数字で証明」
- → どこにもない、**観察を肯定する**チャンネル
