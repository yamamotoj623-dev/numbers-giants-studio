## 1. 設計の核心

### 1.1 もえかが代弁する「3つのカドが立つ本音」

中年巨人ファンの心の中にあるが言いにくい本音を、3 種類に分類:

| 種別 | 本音の内容 | 言いにくさの理由 |
|---|---|---|
| **慢性癖の苛立ち** | 「またこのパターン…ズルズル行く癖どうにかならんのか」 | 自分のXで書くと「批判ばかり」「アンチ」と言われる |

もえかはこれを「**数原さんへの相談 / 観察 / 疑問形**」の形で柔らかく代弁する。

### 1.2 3 つの変換テク

@kandume92 の鋭い男性語を、もえかの「鋭いが品のあるコアファン女子」に変換する 3 原則:

**① 核心キーワードはそのまま採取**

**② 男性語尾だけ翻訳して敬語の rhetorical question に**
- ❌「やれよ」「だぜ」「お前ら」「噛み付いていけ」
- ✅「〜なんですかね…?」「〜じゃないですか?」「〜してほしいです」「〜どうにかならないんですか…?」

## 2. 「ヤバい」運用ルール (★最重要)

`HANDOVER §3 NGワード` では「ヤバい」は元来 NG だったが、@kandume92 流の生々しさの核心ワードのため、**もえかちゃん限定で解禁**する。ただし以下の運用ルール厳守:

### 2.1 解禁条件

| 項目 | ルール |
|---|---|
| 対象キャラ | **もえか (B) のみ**。数原 (A) は引き続き使用禁止 |
| 動画あたりの上限 | **1-2 回まで** (連発するとチープ化、漏れ出る希少性が消える) |
| 使用シーン | **感情ピークの瞬間のみ** (異次元の数値・想定外のプレー・絶望的な悲報) |
| 文型 | 必ず**疑問形 + 敬語**で組み合わせる: `〜ヤバくないですか…?` `これヤバすぎませんか?` |

### 2.2 「出ちゃった感」を作る前後文脈

「ヤバい」は **直前に冷静な観察を置く**ことで「思わず漏れ出る」感を演出する:

❌ **悪い例 (いきなりヤバい)**
```
B: 「平山選手の足、ヤバくないですか?」
```

✅ **良い例 (冷静→漏れ出る)**
```
B: 「数原さん、平山選手…右打者でショートゴロをセーフにしてました」
A: 「これは速さの次元が違いますね」
B: 「これって…ヤバくないですか?」  ← 観察 → 解説 → 漏れ出る
```

### 2.3 NG パターン

- ❌ 連発: 「ヤバい!」「ヤバすぎ!」「もうヤバい!」(1本に2回以上)
- ❌ 言い切り: 「これヤバいです!」(疑問形必須)
- ❌ カジュアル全開: 「やばっ」「やばたん」「マジでヤバい」
- ❌ 数原が使う: A は絶対敬語、ヤバいは品が崩れる

`character-bible.md §4.2` のルール継続。`id:2-5` で両方向の呼びかけ必須:
- A → B: 「もえかちゃん」呼び 1 回以上
- B → A: 「数原さん」呼び 1 回以上

辛口側の rhetorical question は「数原さん…」を頭につけるパターンが多いため、**呼び合いルールとも親和性が高い**。

## 6. 運用上の注意

### 6.1 1 動画あたりのバランス目安 (28-30 scripts 中)

| カテゴリ | 推奨個数 | 備考 |
|---|---|---|
| 共感型 (3.2) / 感心型 (3.4, 3.8) | 8-10 個 | 朗報サイドの土台 |
| 質問型 (3.5, 3.6, 3.8) | 5-7 個 | 数原への振り |
| 辛口型 (3.1, 3.3) | 2-4 個 | 山場で集中投下 |
| 異論型 (3.3, 3.5) | 1-2 個 | 議論誘発、控えめ |
| 漏れ出し型 (3.7, ヤバい) | **1-2 個まで** | ★絶対上限★ |

### 6.2 動画タイプ別のチューニング

| 動画タイプ | 推奨カテゴリ |
|---|---|
| 完敗・連敗振り返り | 3.1, 3.3, 3.5 中心 (辛口側の比重大) |
| 若手特集 | 3.2, 3.4, 3.7 中心 (絶賛 + 漏れ出し) |
| 投手分析 | 3.5, 3.6, 3.8 中心 (起用疑問 + 観察) |
| データ深掘り | 3.4, 3.8 中心 (去年比較 + 受け答え) |
| 編成・系譜論 | 3.2 #10, 3.5 #25, 3.6 中心 |

### 6.3 セルフチェックリスト

台本完成後、以下を確認:

- [ ] 辛口型のサンプルを使う場合、必ず**疑問形 + 敬語**になっているか
- [ ] 中堅・ベテランへの批判は**層 (中堅勢 / ベテラン勢 / 結果が出ない選手)** で語られ、**個人名は出ていない**か
- [ ] 若手称賛は**固有名詞**で具体的か
- [ ] 「ヤバい」は **1 動画 1-2 回まで**、**疑問形**になっているか
- [ ] 「ヤバい」の直前に **冷静な観察文**が置かれ、「漏れ出る」感が作れているか
- [ ] 数原 (A) の口調が**絶対敬語**を維持しているか (もえかの辛口に引きずられていないか)
- [ ] NG ワード (期待したい / 応援したい / 頑張れ / 本当の / 現在地 / 可能性 / 驚愕の) が混入していないか
